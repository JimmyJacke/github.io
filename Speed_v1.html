🌐 Problema de conexão?
• Verifique se está com Wi-Fi ou dados móveis ativos
• Em Android 6, permita acesso à rede para este app em:
Configurações → Apps → [Este App] → Dados móveis → Permitir
🔧 Como permitir o microfone no Android 6:
1. Configurações → Apps
2. Encontre este app
3. Permissões → Microfone → Permitir
Depois toque em "🔄 Já permiti!" acima
